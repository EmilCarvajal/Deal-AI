Los objetos siguen las proporciones adecuadas, pero NO ESTÁN ESCALADOS al 
tamaño correcto, por tanto consultar proporciones si se quieren implementar.

Objetos como las cartas o las fichas son para representar la forma, el valor
de cada uno se representará después individualmente. En caso de usar Coppelia
se aplicará una 'Quick Texture' para añadir el valor de la ficha o de la carta.
